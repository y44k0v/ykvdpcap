## Installation

Use the devtools package:


```r
library(devtools)
install_github("ykv001/ykvdpcap")
library(ykvdpcap)
```

## Functions

This package includes the following functions

* eq_clean_data
* eq_create_label
* eq_data_read
* eq_location_clean
* eq_map
* eq_time
* geom_timeline
* geom_timeline_label
